## Featured Projects

### 🏭 UNIECO — B2B WordPress Refactoring
> [uni-eco.com](https://uni-eco.com)

A full site overhaul for a B2B industrial company. Started with a bloated, outdated codebase and delivered a fast, secure, custom-built WordPress theme.

**What I did:**
- Built a **custom WordPress theme from scratch** — no page builders, no bloat
- Reduced plugins from **20** to **8** without losing any functionality
- Migrated content management from **ACF** to **Carbon Fields**
- Implemented **Custom Post Types (CPT)** for production catalogue sections
- Pixel-perfect **responsive & cross-browser layout** from Figma
- Hardened security: custom admin URL, removed vulnerable plugins
- Updated PHP & WordPress core to current versions

**Result:** Google PageSpeed **90–95+ / 100** ✅

### ♻️ EKOPRAIM — B2B Bitrix Site from Scratch
> [ekopraim.ru](https://ekopraim.ru)

Full-cycle development of a commercial B2B site on **1С-Bitrix: Small Business**.

**What I did:**
- Responsive & cross-browser layout from Figma mockup
- Designed **infoblocks architecture**, smart filter, and SEO-friendly URLs
- Built **multiregional support** via subdomains
- Wrote a custom **Cron script** to auto-generate unique `sitemap.xml` per subdomain
- Developed a custom **Excel import/export module** for product catalogue
- Integrated **Битрикс24**: automatic lead collection & routing
- Passed **cart contents directly into Bitrix24** leads, cutting sales team response time

**Result:** Google PageSpeed **85–90+ / 100** on Bitrix ✅
